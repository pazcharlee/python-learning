# Chapter 2 – Working with Variables and Input

## Overview

This chapter builds on the basics introduced in Chapter 1 by adding interactivity and data handling. The focus is on storing information in variables, accepting user input, and using that input to influence program behavior. These concepts form the foundation for more dynamic Python programs.

## Topics Covered

* Creating and using variables
* Understanding data types (strings and integers)
* Using the `input()` function to get user input
* Type conversion with `int()` and `str()`
* Combining input, variables, and output in simple programs

## Skills Developed

* Storing and updating values using variables
* Writing programs that respond to user input
* Converting and working with different data types
* Building simple interactive Python scripts
